# ds-event-stream-py-sdk

A Python package from the ds-common library collection.

## Installation

Install the package using pip:

```bash
pip install ds-event-stream-py-sdk
```

Or using uv (recommended):

```bash
uv pip install ds-event-stream-py-sdk
```
